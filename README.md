# 7DSGC Offsets
This repository is intended for educational and research purposes only. It contains offset references and notes related to process structure analysis for learning how software is organized in memory.

It is not intended to be used as a runtime tool or for modifying any software.

---

## Overview
This project focuses on studying and documenting internal structures such as offsets and memory layouts for educational understanding of how applications function at a low level.

The goal is to help developers learn reverse engineering concepts in a controlled and theoretical context.
